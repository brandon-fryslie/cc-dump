# Theme Variable Comparison (All Themes, One Hex Per Cell)

Fully expanded comparison: one variable per row, one hex value per theme cell.

Normalization applied:
- `ansi_*` -> hex via Textual color parser
- `ansi_default` -> deterministic fallback (`foreground`/`background`-aware)
- `transparent` and `initial` -> `#00000000`
- `auto NN%` -> contrast color against theme background with alpha `NN%`
- `#RRGGBB NN%` -> converted to `#RRGGBBAA`

Style-only variables were excluded because they are not colors.

- Themes: 18
- Raw common variables: 163
- Hex-convertible rows: 151
- Excluded non-color variables: 12
